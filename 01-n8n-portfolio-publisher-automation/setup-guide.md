# Setup Guide - n8n Portfolio Publisher Automation

## Prerequisites

- n8n installed and running
- Basic understanding of n8n nodes
- GitHub repository for portfolio projects

## Steps

1. Open n8n.
2. Create a new workflow.
3. Add a Manual Trigger node.
4. Add a Set node for project details.
5. Add a Code node to generate portfolio files.
6. Run the workflow.
7. Review the generated documentation output.

## Notes

This setup guide is generated automatically by the n8n Portfolio Publisher workflow.
